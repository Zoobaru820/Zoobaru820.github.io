Role: Lead Developer & UX Designer

Tools: HTML, CSS, JavaScript, Figma

Timeline: Fall 2024

Problem

Schmeeckle Reserve needed a non-invasive way to monitor bat populations as part of their conservation efforts.

Solution

I helped develop an interface for a motion-triggered camera system (BatCam) that captures and stores bat activity. The UI focused on clarity, accessibility, and ease-of-use for staff with minimal tech experience.

My Contributions

  • Designed a minimalist interface that functions well in low-light conditions.
  • Created custom CSS animations to simulate bat detection feedback.
  • Created a design and style sheet in Canva to be used by future developers.
